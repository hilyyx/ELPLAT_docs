# 📊 Автоматический обработчик QR-кодов из Excel (ГОСТ-платежи)

Удобный скрипт на Python для массового извлечения, распознавания и переноса данных из графических QR-кодов (стандарта ГОСТ) в итоговую Excel-таблицу по лицевым счетам.

---

## 🚀 Основные возможности

- **Массовая обработка** — автоматический обход всех изображений в исходном Excel-файле.
- **Умная предобработка изображений:**
  - удаление прозрачности (альфа-канала) и перевод изображения на белый фон;
  - масштабирование и добавление полей (*Quiet Zone*) для повышения качества распознавания OpenCV.
- **Декодирование ГОСТ-кодировок** — корректное восстановление кириллицы (`latin1 → cp1251`).
- **Быстрый поиск (кэширование)** — мгновенное сопоставление данных по лицевому счету (столбец **A**) без повторного перебора строк.
- **Устойчивость к ошибкам** — если изображение не является QR-кодом или повреждено, обработка продолжается, а ошибка фиксируется в итоговой статистике.

## 📋 Заполняемые поля

Скрипт ищет строку по совпадению **Лицевого счета** (столбец **A**) и заполняет следующие поля:

| Столбец | Поле Excel | Источник из QR | Пример |
|:-------:|------------|----------------|---------|
| **A** | Лицевой счет | `persAcc` *(используется для поиска)* | `26214099215` |
| **B** | Паспорт | *(заполняется заранее)* | `4512 ******` |
| **C** | Группа | `Purpose` *(извлекается подстрока «гр. X»)* | `гр. 1` |
| **D** | ФИ ребенка | `childFio` *(Фамилия Имя)* | `Иванов Иван` |
| **E** | Группа (дубль) | `Purpose` | `гр. 1` |
| **F** | Район | *(фиксированное значение)* | `Устиновский район` |
| **G** | Код услуги | `PayeeINN + "_1"` | `1831034444_1` |
| **H** | Код (КБК) | `CBC` *(последние 4 цифры)* | `1010` |
| **I** | ОКТМО | `OKTMO` | `94701000` |
| **J** | Сумма за месяц | `Sum` *(копейки → рубли)* | `127.85` |

## 💻 Запуск

1. Поместите:
   - `main.py`;
   - исходный Excel-файл с QR-кодами;
   - итоговую таблицу `Итоговый.xlsx`

   в одну папку.

2. Запустите скрипт:

```bash
python main.py
```

3. После завершения обработки в консоли появится подробный лог работы и итоговая статистика.
